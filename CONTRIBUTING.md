# Contributing to kenya-a2a

Thank you for your interest in contributing to kenya-a2a!

## How to contribute

This repository accepts **bug reports** and **feedback** via GitHub Issues.

For feature requests or questions, please [open an issue](../../issues).

must align with that mission.
